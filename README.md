This project showcases the application of Python in data analytics through two distinct case studies. The first part involves survival analysis using the Titanic dataset, where features like age, gender, and passenger class were examined to uncover patterns influencing survival rates. Techniques such as exploratory data analysis and classification models were applied to derive meaningful insights.
The second part focuses on agricultural data, where crop recommendations were generated based on region, season, soil, and irrigation type. Additionally, crops were prioritized for government subsidy using factors like yield, market price, and export potential. The project demonstrates how Python can be effectively used for data-driven decision-making across diverse domains.
